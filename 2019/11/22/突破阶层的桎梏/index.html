突破阶层的桎梏

发表于 2019-11-22 分类于 Others 1.6k

起

这个标题起得有点大，对吧。

阶层，是个什么概念？活了这么些年，想必你我都清楚，阶层不是那么容易突破的。但是，它绝不是不可突破。

我为什么突然想到写这篇文章？缘由，是我看见我妈在手机上刷单，每天都花很多时间在上面，我说了好几次让她用电脑，因为用电脑比用手机效率高。今天我看她在那忙我又叫她用电脑，电脑基本操作学习资料我早已给她找好了，结果她说了一句在我意料之外却也是情理之中的话：“用电脑花时间，要现学”。

瞬间无语。

其实就这种问题说过很多次了，我今天算是想明白了。

我想，这就是为什么大多数人一辈子平庸的原因。

永远做底层劳动力，玩也不会玩，看着心急。

计算机就在那，环境都配好了，资料也找好了，就只需要学习一下，提高效率。她觉得学计算机基础操作麻烦，为什么就不觉得每天费那么多时间在手机上麻烦呢？这远不是挣不挣钱的问题，这折射出了一种基本的思维定势，定势，就定死了。

我觉得，明明有使用计算机的条件，却不会用计算机去发现广袤的新世界，真的是人生的遗憾。

我忽然就明白了鲁迅所说的：“哀其不幸，怒其不争”。其实在平常人看来，好像没什么不幸的：家庭还可以，儿子也考上大学了，哪里不幸了？可是在我看来，一个人停止了进步，就是对他/她自己最大的不幸。

很多人说，年龄大了，思维就固定了。我呸！神经元具有终身可塑性！特别是中年过后，别人都原地踏步了，你多学一点点就能超过很多人，比青年时期的竞争小多了。哪怕每天学半小时的最基本的计算机知识、或者学厨艺、或者学英语、学任何你想学的都可以，每天就花那么半小时都能日积月累超出别人一大截。

  • 一年365天，每天退步1%，一年之后累计剩下多少呢？ $(1-0.01)^365$

转

为什么？因为平庸之人只会在无意义的重复劳动和浅层次的消遣娱乐之间蹉跎，这两者都不会拓展你的人生。这就是为什么两个差不多的年轻人，几十年后一个于明镜高堂之上闲庭信步，一个于贫无立锥之地庸庸碌碌。

我是在说普遍现象，不是特指谁，从普遍现象推导得出普遍适用的结论，并以此指导自己的人生。

大家有一个共识，那就是 如果你有一个真的朋友，那他/她就应该指出你的缺点，批评你的错误，而不是看着你走下坡路。我觉得，一个好的子女也应该勇于指出父母的缺点和错误。更何况，除了你，还有谁会指出你父母的缺点？父母听不听是一回事儿，至少你应该不卑不亢地 语气稍加委婉地 说出来。

当然，如果你真的觉得你父母无药可救，不说也罢。

那么，现在来探讨一下原因：为什么迷茫于方寸之间了呢？

绝不是因为年龄的原因，而是因为内心缺乏深层次的源动力。

为什么缺乏动力？在没有天灾人祸的前提下，可能是因为没有找到自己真正热爱的事物，没有那一份热忱。究其原因，可能应试教育和“中国式家长”教育得背锅。再说下去就是中国教育体系和传统文化的问题了，就此打住。

合

如何突破阶层的桎梏？格局、智慧、魄力，缺一不可。

中国人口基数这么大，内卷这么严重，不是说上升就能上升的。
